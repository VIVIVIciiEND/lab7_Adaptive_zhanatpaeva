# Лабораторная работа №8. Адаптивная верстка: Grid
*Цель работы:* освоить современные технологии CSS-разметки — Grid.
*Адаптивная верстка* - это подход к созданию веб-страниц, при котором дизайн 
и контент автоматически подстраиваются под размер экрана устройства
*Grid* - двумерная система компоновки для создания сложных сеток.

## Сравнение Flexbox и CSS Grid
|Критерий|Flexbox|CSS Grid|
|---|---|---|
|Тип компоновки|Одномерная|Двумерная|
|Управление|Строка или столбец|Строки и столбцы|
|Основное назначение|Выравнивание элементов|Построение макетов|
|Подходит для|Компонентов|Страниц и сеток
|
